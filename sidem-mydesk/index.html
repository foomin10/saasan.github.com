SideM風会話ジェネレータ

Tweet

アイドルマスター SideM にある、マイデスクでアイドル同士が会話するシステム。あれが シンデレラガールズ や ミリオンライブ！ にもあったらなぁ…。ということで作ってみた。

遊び方

  1. 名前と台詞を入力します。
  2. 各コマに画像ファイルをドラッグアンドドロップで入れます。
    画像ファイルは、ブラウザーで表示中のファイルやパソコンに保存されているファイルが使えます。
  3. 各コマをドラッグして画像の位置を調整します。
  4. 各コマの上でマウスホイールを動かして画像を拡大/縮小します。
  5. 完成したら「保存用画像を出力する」ボタンを押します。
  6. 出てきた画像を右クリックして保存します。
  7. 画像とハッシュタグ「#SideM風会話」を付けてツイートすると作者が喜びます！

動作環境

  • Internet Explorer 11、Mozilla Firefox 27、Google Chrome 33で動作する事を確認しています。
  • Internet Explorer 9で動作しない事を確認しています。
  • 2014/03/09 0:19 Mozilla Firefox 以外で保存できない問題を修正
